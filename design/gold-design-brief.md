# Промт: золотий luxe-редизайн запрошення «Вісточка»

Для claude.ai/design (або іншого UI-генератора). Це редизайн існуючого
data-driven односторінкового весільного запрошення: ті самі секції, змінюється
лише візуал. Мова контенту українська. Золото симулювати CSS-градієнтами
(жодних растрових зображень фольги).

---

## Напрямок
Modern Luxe Gold (editorial). Стримана розкіш: багато повітря, велика серифна
типографіка як головний акцент, золото як «фольга», тонкі золоті лінії,
контрастний ТЕМНИЙ hero. Без кітчу і важких орнаментів. Сучасно, дорого, чисто.

## Палітра (точні hex)
Світла база:
- bg (фон сторінки): #FBF7EF (тепла слонова кістка)
- surface (секція): #F4ECDD (шампань)
- surface-low (світліша секція): #FAF3E6
- ink (основний текст): #211C16
- ink-2 (вторинний текст): #4A4135
- muted (підписи): #8A7E6B
- line (тонкі лінії, розділювачі): #E2D5BD

Золото:
- gold-gradient (для заголовків, імен, іконок, розділювачів):
  linear-gradient(135deg, #B38728 0%, #DBA845 18%, #F7E08A 38%, #C99A3A 60%, #FBF1B8 78%, #A9772C 100%)
- gold-accent (суцільний, для кнопок/посилань): #C2A878
- gold-deep (для золотого ТЕКСТУ на світлому, щоб був читабельним): #9A7B3F

Темний контраст (hero, footer, опційно CTA-секція):
- ink-dark (фон): #16130F (тепла-чорна)  // knob: смарагд #0E2A22 або navy #10182E
- текст на темному: #F4ECDD; золотий градієнт на темному сяє найкраще.

## Типографіка (Google Fonts, ОБОВʼЯЗКОВО з кирилицею)
- Display (заголовки, імена пари): Playfair Display або Cormorant Garamond.
  Великий, editorial; імена пари дуже великі. Ваги 400/500/600.
- Body (абзаци, підписи): Inter або Manrope, ваги 300-500. Дрібні підписи капсом
  із широким letter-spacing.
- Опційний рукописний акцент (Great Vibes) лише на ОДИН елемент (напр. «&» або
  «Save the date»), не для тексту.

## Золото як «фольга»
- Заголовки секцій та імена пари: золотий градієнт через background-clip:text.
- Розділювачі: тонка золота лінія 1px із маленьким ромбом/крапкою по центру
  (опційно міні art-deco промінь). Стримано.
- Іконки (контакти, локації): тонкий золотий stroke.
- Кнопки: суцільний gold-accent із темним текстом; на темному фоні золотий контур.
- Золото = акцент на ~10-15% площі, решта повітря і нейтраль.

## Мотиви / деталі
- Лінійний hairline-стиль. Гострі кути (editorial) або ледь скруглені 2-4px.
- Опційний делікатний art-deco акцент: тонка геометрична рамка hero, промениста
  міні-зірка у розділювачах. Дуже стримано.
- Ледь помітна матова текстура (паперова/льняна) на ТЕМНИХ секціях, щоб золото
  «лежало» на матовому.

## Структура секцій (як у рушії, зверху вниз)
0. Лендинг бренду (корінь /): назва «Вісточка», підзаголовок, блок контактів із
   золотими лінійними іконками (telegram / instagram / email), підвал.
1. Nav: монограма/імена, тонке золоте підкреслення, лінки (Програма, Локації),
   RSVP як золота кнопка.
2. Hero (ТЕМНИЙ): «Ми одружуємось», великі імена пари золотим градієнтом,
   золотий «&», дата, стрілка-скрол вниз. Можлива тонка золота рамка.
3. Countdown: «До весілля лишилось», великі золоті цифри, підписи капсом muted.
4. Love story: заголовок золотом, текст + портретні фото, тонкі золоті лінії.
5. Галерея: асиметрична сітка, тонка золота межа при ховері, lightbox.
6. Таймінг: вертикальна золота лінія, час золотим, подія серифом.
7. Дрес-код: свотчі палітри (золото + нейтраль), заголовок золотом.
8. Локації: картки церемонія/банкет, золота кнопка «Побудувати маршрут».
9. RSVP: картка, поля з тонкою золотою лінією знизу при фокусі, золота кнопка
   «Відправити».
10. Telegram: заклик у чат, золота кнопка-контур.
11. Footer (ТЕМНИЙ): «Святкуйте з нами!», монограма золотом, дискретний підпис.

## Технічні вимоги
- Mobile-first, брейкпоінти ~600/900, усе адаптивно.
- Веб-рендер, без растрової фольги: золото лише CSS-градієнтами.
- Контраст: золотий ТЕКСТ на світлому робити gold-deep (#9A7B3F) для читабельності;
  яскравий градієнт лише на темному або на великих декоративних заголовках.
  Дотримати WCAG AA для основного тексту.
- prefers-reduced-motion вимикає анімації; font-display: swap; системні фолбеки.
